# Supershop-Data-Analysis(intractive Dashboard creation using MS Excel)

## Project Overview
This project analyzes Superstore 2009,2010,2011,2012 sales data to uncover business insights and sales trends.
## Data set used
- <a href="https://github.com/nazmul23423/Data-Analysis-Excel-project-First-Project-/blob/main/Sample%20-%20Superstore%20Sales%20(Excel).xlsx">Dataset</a>

## Business Questions / Analysis Objectives
- What is the total sales generated each year?
- Which product category generates the highest sales?
- Which region performs best in terms of profit?
- What are the yearly sales growth trends?
- Which sub-category contributes the most revenue?
- Which products have the highest and lowest sales?
- How do sales and profit vary across different regions?
- What are the key KPI metrics of the business?
- Which customer segments generate the highest profit?
- What business insights can improve future sales performance?
